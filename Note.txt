Starting
--------------

1. Open CMD
2. Check is node istalled, "node -v" (cmd prompt)
3. If not then Install Node js

4. Let's start
5. Write "npx create-react-app Any_Name" (cmd prompt)



Run
---------
1. open CMD in the project folder (your newly created folder) 
2. "npm start" (cmd prompt)

close when the cmd turn off
